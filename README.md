# Colburn Auctions

Business website for Colburn Auctions — an estate and antique auction house based in Connecticut.

**Live site:** [colburnauctions.com](https://colburnauctions.com)

## What this project demonstrates

Full-cycle client delivery: design, build, domain registration, DNS configuration, hosting, and business email setup — not just the code.

- Custom Next.js site built to the client's brand (color palette, tone)
- Domain purchased and configured through Cloudflare (DNS, SSL/CDN)
- Deployed and hosted on Vercel
- Business email (Zoho Mail) configured with SPF and DKIM authentication for deliverability

## Stack

Next.js (App Router) · TypeScript · Tailwind CSS · shadcn/ui components · Vercel

## Structure

```
app/            Route and page composition
components/     Page sections (hero, header, footer, categories, etc.)
components/ui/  Shared UI primitives
public/         Static assets
```

## Status

Initial version live. A simplified content revision is in progress based on client feedback, incorporating the owner's real background and current auction details.
